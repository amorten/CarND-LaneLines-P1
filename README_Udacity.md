<h2><a id="user-content-overview" class="anchor" href="#overview" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Overview</h2>
<p>When we drive, we use our eyes to decide where to go.  The lines on the road that show us where the lanes are act as our constant reference for where to steer the vehicle.  Naturally, one of the first things we would like to do in developing a self-driving car is to automatically detect lane lines using an algorithm.</p>
<p>In this project you will detect lane lines in images using Python and OpenCV.  OpenCV means "Open-Source Computer Vision", which is a package that has many useful tools for analyzing images.</p>
<p>To complete the project, two files will be submitted: a file containing project code and a file containing a brief write up explaining your solution. We have included template files to be used both for the <a href="https://github.com/udacity/CarND-LaneLines-P1/blob/master/P1.ipynb">code</a> and the <a href="https://github.com/udacity/CarND-LaneLines-P1/blob/master/writeup_template.md">writeup</a>.The code file is called P1.ipynb and the writeup template is writeup_template.md</p>
<p>To meet specifications in the project, take a look at the requirements in the <a href="https://review.udacity.com/#!/rubrics/322/view">project rubric</a></p>

<p>For this project, a great writeup should provide a detailed response to the "Reflection" section of the <a href="https://review.udacity.com/#!/rubrics/322/view">project rubric</a>. There are three parts to the reflection:</p>

<p>All that said, please be concise!  We're not looking for you to write a book here: just a brief description.</p>
<p>You're not required to use markdown for your writeup.  If you use another method please just submit a pdf of your writeup. Here is a link to a <a href="https://github.com/udacity/CarND-LaneLines-P1/blob/master/writeup_template.md">writeup template file</a>.</p>
